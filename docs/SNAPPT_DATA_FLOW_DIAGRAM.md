## Appendix: Glossary

**Applicant** - Individual applying for rental property
**Entry** - A screening application submission
**Property** - Rental property/community
**Proof** - Document uploaded for verification
**Fraud Score** - ML-based document authenticity score (0-100)
**Bank Linking** - Direct bank account connection for verification
**IDV** - Identity Verification
**PMS** - Property Management System (e.g., Yardi)
**Webhook** - HTTP callback for real-time event notifications
**DLT** - Delta Live Tables (Databricks ETL framework)
**RBAC** - Role-Based Access Control
